# 2. Add Two Numbers

这是一个很简单的算法，直接暴力枚举即可，但是要注意的是，数字相加可能产生进位，所以需要考虑进位的可能



将两个数字相加，然后和10取整即可，如果数字是俩位数，就会产生进位，然后将进位数保存下来，在和10取余，就能算出来这个位置的数
